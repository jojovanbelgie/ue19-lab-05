# UE19 - Lab 05 : API Joke Fetcher

## Description
Cette application interroge l’API publique **JokeAPI** pour afficher une blague aléatoire dans le terminal.

## Installation
pip install -r requirements.txt
### Prérequis
- Python 3.11+
- Git
- Docker (optionnel)
## lancement sans ou avec docker
- python app.py
- docker build -t joke-app
- docker run --rm joke-app
